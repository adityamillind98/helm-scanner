About

This site is provided to easily search and view the infrastrucrutre as code “Blast Radius” graphs, produced by the Helm-Scanner OSS codebase.

More information

Find the codebase on Github or carry on the conversation on our #CodifiedSecurity Slack profiles.

Made by the Bridgecrew.io DevRel team

Site built with Hugo using helm-scanner generated html graphs, theming and template based heavily on Verummeum by PhilipVis with thanks for an awesomely simple OSS site with working indexable search!
